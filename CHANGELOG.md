### Features

- Initial release of AMD Windows AI Toolkit master hub
- `doctor.py` — comprehensive environment health check (GPU, drivers, DX12, Vulkan, all backends)
- `setup_env.py` — interactive one-time setup wizard with conflict detection
- `check_gpu.py` — quick GPU status snapshot (no dependencies)
- `run.bat` — quick launcher menu for all scripts
- Full ecosystem table covering 14 AMD Windows AI repos
- Backend decision tree (DirectML vs Vulkan by use case)
- VRAM quick reference table (4GB → 24GB)
- Driver/software requirements table
- Common issues and fixes documentation
